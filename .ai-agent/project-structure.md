# Project Structure

# Folders in src/

- assets/: Game assets (levels.json)
- ts/: TypeScript source code
  - entities/: Game entity classes
  - __tests__/: Unit tests for TS files

# Other folders

- .ai-agent/: AI agent rules and workflows
- .kilocode/: Kilo Code configuration and Memory Bank
- docs/: Documentation files
- tests/: Test setup and global mocks
