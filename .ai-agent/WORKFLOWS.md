# Intro

This file contains the workflows that all AI agents must follow. Each workflow is defined in a separate file

## General Guide

AI agents must consider all the workflows listed below before starting to work.
When the AI agent need to handle a work required by the user, it must formulate the correct tiny and detailed steps to follow, by analyzing all the workflows. This mean to generate a internal TO-DO list which contains in correct order and combination the steps of the differents workflows.
