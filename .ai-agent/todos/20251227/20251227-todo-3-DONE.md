# TODO: Fix Level Progression Crash

## Problem Description
The game crashes or freezes when transitioning from Level 1 to Level 2.

## Technical Analysis
1.  **Sparse Config Failure**: [`src/assets/levels.json`](src/assets/levels.json) defines Level 2 without `rows`, `cols`, `speed`, or `enemyCount`.
2.  **Inheritance Bug**: The [`resolveLevelConfig`](src/ts/Game.ts:135) method in [`src/ts/Game.ts`](src/ts/Game.ts) incorrectly resets the configuration object when it encounters a new static level definition, instead of merging it with the existing state.
3.  **Resulting NaN**: Level 2 initializes with `undefined` values, leading to `NaN` calculations in [`EnemyWave.ts`](src/ts/entities/EnemyWave.ts) and stopping the game loop.

## Proposed Fix
Refactor [`resolveLevelConfig`](src/ts/Game.ts:135) to properly accumulate properties from all levels up to the target level, ensuring that sparse levels correctly inherit parameters from their predecessors.
